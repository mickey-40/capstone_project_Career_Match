# AI Resume Analyzer (CareerMatch AI)

A full-stack capstone project for WGU D424 — Software Engineering Capstone.  
This project implements an AI-driven resume analyzer that compares a user’s resume against a job description and provides tailored feedback.

---

## 🧱 Architecture Overview

**Frontend:** Android (Kotlin, Jetpack Compose, Room, Retrofit)  
**Backend:** Python (FastAPI, SQLAlchemy, Pydantic)  
**Database:** SQLite (development) / PostgreSQL (production)  
**Containerization:** Docker  
**Hosting:** Google Cloud Run (planned deployment)
